Some useful commands:
- git branch -a (current branch is starred, `-a` flag shows remotes )
- git switch <option> <branch> (alternative to git checkout <branch>)
- git add <file/directory> && git commit -m "<message here>"
- git commit --amend "<message here>" (amends most recent commit message, ?if not pushed to remote?)
- git cherry-pick
- git config --list --show-origin --show-scope
- git log <branch>..<branch>
- git log --graph --oneline --all
- git worktree add <dir> -b <branch name>
- git blame 
- git stash && git stash pop
